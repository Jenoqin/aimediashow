# AI × 传媒变革 首页设计文档

## 项目背景

为 HW 公司设计一个面向传媒行业从业人员的静态 Demo 网站首页。目标是技术引领式交流——展示 AI 新技术对传媒行业工作方式及行业生态带来的变革，而非推销具体产品。不呈现公司标识。

## 设计目标

1. **技术引领**：以行业观察者和技术推动者的视角呈现内容
2. **专业可信**：数据支撑，场景具体，让传媒从业者产生共鸣
3. **方便扩展**：首页每个板块后续可通过链接深入介绍细节
4. **响应式**：适配桌面端和移动端

## 整体架构

单页滚动式布局，通过锚点导航跳转到各板块。

```
[固定导航栏] — 锚点链接到各板块
  │
[Hero 首屏] — 全屏，大标题 + 核心数据
  │
[选题策划] — 板块 1（橙红色调 #f97316）
  │
[内容生产] — 板块 2（青色调 #00d4ff）
  │
[审核校对] — 板块 3（紫色调 #7c3aed）
  │
[智能分发] — 板块 4（金色调 #f59e0b）
  │
[数据洞察] — 板块 5（绿色调 #10b981）
  │
[未来展望] — 板块 6（粉紫色调 #ec4899）
  │
[CTA 区域] — 引导深入交流
  │
[Footer] — 简洁版权信息
```

## 视觉风格

- **主题**：深色科技风
- **背景色**：`#050810`
- **主文字**：白色 `#fff`
- **次要文字**：`#8899bb`
- **辅助元素**：半透明卡片 + 细边框 + hover 发光效果
- **背景纹理**：40px 间距的淡青色网格线 + 径向渐变光晕
- **字体栈**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "PingFang SC", "Microsoft YaHei", sans-serif`

## 页面结构

### 1. 导航栏

固定顶部，半透明深色背景 `rgba(5,8,16,0.9)` + backdrop-filter 模糊。

- **左侧**：Logo 图标（渐变圆角方块 + 文字"AI × 传媒变革"）
- **中间**：导航链接 — 首页 / 选题策划 / 内容生产 / 审核校对 / 智能分发 / 数据洞察 / 未来展望
- **右侧**：CTA 按钮"联系我们"（渐变胶囊形）

### 2. Hero 首屏

全屏高度（100vh），居中布局。

- **背景**：深色底 + 三色辐射光晕（cyan/purple/orange 位置不同）+ 半透明网格线（40px 间距）
- **顶部标签**：胶囊形边框标签 "2026 · 传媒行业 AI 变革观察"
- **主标题**："AI 正在重塑 / 传媒行业的每一个环节"，"每一个环节"用 cyan→purple 渐变
- **副标题**：从选题策划到内容分发，从事实核查到趋势预判 / 探索人工智能技术如何重新定义媒体人的工作方式
- **双 CTA 按钮**：
  - 主按钮"探索六大变革场景"（渐变背景+发光阴影）
  - 副按钮"下载行业观察报告"（透明边框）
- **底部数据指标条**：4 列等宽，显示关键数据
  - 78% 头部媒体已引入 AI 工具（青色）
  - 3.2× 内容生产效率提升（紫色）
  - 96% 智能分发精准度（金色）
  - 50% 运营成本降低（绿色）
- **滚动提示**：底部 "向下滚动探索 ↓"，pulse 动画

### 3. 场景区块（×6）

6 个场景区块采用统一模板，居中卡片网格布局，仅切换主题色。

每个区块包含三层：

**第一层 — 标题区**（居中）：
- 顶部装饰线（主题色渐变）
- 图标 + 场景标签（大写字母间距）
- 主标题（关键词用主题色高亮，`<em>` 标签）
- 描述文字

**第二层 — 功能卡片**（3 列居中）：
- 每个卡片：图标 + 标题 + 描述
- 卡片背景：主题色 5% 透明度
- 卡片边框：主题色 15% 透明度
- hover 效果：边框加亮 + 轻微上移 + 阴影

**第三层 — 数据亮点**（横排指标条）：
- 3 个数据指标横排
- 数字用主题色，大号粗体
- 标签文字 `#667`

#### 3.1 选题策划（orange #f97316）

- 标题：让 AI 成为你的 *首席情报官*
- 描述：实时追踪全网热点，预测话题趋势，智能推荐最优选题方向 / 告别"拍脑袋"决策，让数据驱动每一个选题
- 功能卡片：热点追踪 / 趋势预测 / 选题推荐
- 数据：5min 热点发现速度 / 92% 趋势预测准确率 / 3× 爆款选题命中率

#### 3.2 内容生产（cyan #00d4ff）

- 标题：让 AI 成为你的 *首席编辑*
- 描述：从文字稿件到多媒体内容，AI 覆盖内容生产全链路 / 让创作者专注于思考，将重复劳动交给智能助手
- 功能卡片：智能撰稿 / 多媒体生成 / 多语言适配
- 数据：3.2× 生产效率提升 / 85% 人工干预减少 / 24/7 全天候不间断

#### 3.3 审核校对（purple #7c3aed）

- 标题：让 AI 成为你的 *安全守门人*
- 描述：多维度自动审核，从事实核查到版权合规 / 大幅降低人为疏漏，守护媒体公信力
- 功能卡片：事实核查 / 敏感检测 / 版权合规
- 数据：99.2% 事实核查覆盖率 / 0.3s 单篇审核耗时 / 80% 人工审核工作量减少

#### 3.4 智能分发（gold #f59e0b）

- 标题：每个读者都是 *唯一受众*
- 描述：AI 驱动的个性化推荐引擎，让每一条内容精准触达目标读者 / 实现从"广而告之"到"精准投喂"的转变
- 功能卡片：个性推荐 / 多渠道推送 / 实时优化
- 数据：96% 分发精准度 / 3.2× 触达效率提升 / 50% 用户留存增长

#### 3.5 数据洞察（green #10b981）

- 标题：用数据驱动每一个 *决策*
- 描述：从舆情监测到用户行为分析，从效果归因到趋势预测 / 让数据成为媒体运营的核心引擎
- 功能卡片：舆情监测 / 行为分析 / 效果归因
- 数据：360° 全景洞察 / 实时 数据更新 / 99.9% 系统可用性

#### 3.6 未来展望（pink #ec4899）

- 标题：人机协作的 *新范式*
- 描述：AI 不会取代媒体人，但会用 AI 的媒体人将取代不会用 AI 的媒体人 / 探索未来的工作方式与行业生态
- 功能卡片：人机协作 / 技能进化 / 生态重构
- 数据：2030 全面智能化预期 / 70% 流程将被 AI 辅助 / ∞ 想象空间

### 4. CTA 区

- **背景**：底部辐射光晕 + 网格线（与 Hero 区呼应）
- **装饰线**：居中渐变线（透明→cyan→透明）
- **主标题**："准备好探索 *AI 驱动的媒体未来* 了吗？"（渐变文字）
- **副标题**："与行业先行者一起，探索技术变革的无限可能"
- **双按钮**：
  - 主按钮"预约技术交流"（渐变+发光）
  - 副按钮"下载完整观察报告"（透明边框）

### 5. Footer

单行三段式布局，`1px` 半透明分隔线与 CTA 区隔开。

- **左侧**：Logo（同导航栏）
- **中间**：导航链接 — 首页 / 关于我们 / 联系方式 / 隐私政策
- **右侧**：版权信息 "© 2026 AI × 传媒变革. 技术引领，探索未来."

## 交互动效

- **hover 效果**：功能卡片边框加亮 + 轻微上移（`transform: translateY(-4px)`，`transition: all 0.3s`）+ 阴影
- **导航链接**：hover 变色为主题色
- **按钮**：hover 阴影加深 + 轻微上移
- **滚动提示**：`animation: pulse 2s infinite`
- **数据数字**：静态展示

## 技术栈

- **纯静态**：单个 HTML 文件，内联 CSS
- **零依赖**：无外部 CSS/JS 框架，无图片资源（使用 Emoji 作为图标）
- **响应式设计**：CSS Media Query 适配移动端

## 响应式断点

- 桌面端（>1024px）：完整布局，3 列卡片
- 平板端（768-1024px）：卡片适当缩小间距
- 移动端（<768px）：单列堆叠，导航链接隐藏，数据条换行

| 板块 | 详情页文件名 |
|------|-------------|

| 内容生产 | `content-production.html` |
| 审核校对 | `content-review.html` |
| 智能分发 | `smart-distribution.html` |
| 数据洞察 | `data-insight.html` |
| 未来展望 | `future-outlook.html` |
